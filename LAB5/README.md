# 🚀 MPI Assignment 5 – Blocking vs Non-Blocking Communication

## 📌 Aim

To study and implement **MPI communication techniques**, including:

* Blocking Communication
* Non-Blocking Communication
* Collective Operations (MPI_Bcast, MPI_Reduce)

---

## 🧾 Description

This assignment demonstrates different MPI concepts through 5 programs:

* Parallel computation (DAXPY, Dot Product)
* Communication comparison (Broadcast)
* Master-Slave model (Prime & Perfect numbers)

---

## ⚙️ Requirements

* MPI Library (MPICH / OpenMPI / MS-MPI)
* C++ Compiler (`g++` or `mpic++`)
* Linux / WSL / Windows (with MPI setup)

---

## 🛠️ Compilation

Using Makefile:

```bash
make
```
---

## 📊 Concepts Covered

* Blocking vs Non-blocking Communication
* Deadlock and its prevention
* MPI_Bcast (Broadcasting data)
* MPI_Reduce (Combining results)
* Parallel Speedup & Efficiency

---

## 📈 Learning Outcomes

* Understand parallel programming using MPI
* Learn communication optimization techniques
* Implement real-world distributed algorithms

---


## ✅ Conclusion

This assignment highlights the importance of efficient communication in parallel systems and demonstrates how MPI improves performance through distributed computing.
